一、角色结论卡片

VPP Dispatch

7 个有效项已在 train probe 中可重建。舒适度/SOC 与可用容量支付合计约 89%；私有利润约 10.8%；service/contract/projection 项几乎未激活。

R_dispatch = profit + service + availability - shortfall - projection - comfort/SOC - degradation

VPP Portfolio

7 个窗口项已可重建。当前主要由 switching cost、未来 comfort/SOC 与长期利润代理构成；verified capacity、future projection、future shield 基本未激活。

R_portfolio_window = profit + capacity - shortfall - shield - projection - comfort/SOC - switching

DSO

DSO 公式完整，但审计表不是干净的 algebra decomposition。dso_over_conservative 是诊断基量，dso_curtailment_cost 才是实际扣分项之一；安全裕度父项和子项也不能双计。

R_DSO = U_safe - procurement - loss - curtailment - smoothness - margin - violation - failure - projection
